    Общее:
* Программа подключается к аккаунту Telegram в виде приложения (бота).
* Считывает (в реальном времени) информацию о входящих сообщениях.
* Начинает работу при поступлении "типового" уведомления от "EtherDrops Tracking Bot".

    Перед началом работы:
1. Необходимо зарегистрировать приложение (бота) Telegram. Для этого следуйте инструкциям на:
https://my.telegram.org/auth?to=apps
2. После регистрации Вам будут выданы уникальные коды ("api_id" и "api_hash") Вашего приложения.
3. В скрипте "parserCMC.py" в соответсвующие переменные необходимо записать:
    * Свой ник-нейм в Telegram (пример: @Q_w_E_rty);
    * Полученные при регистрации значения "api_id" (как int) и "api_hash" (как str).
    * Доп. стартовые параметры рекомендуется оставить по умолчанию.
4. Запуск программы. При первом(!) запуске программы необходимо будет:
    * Ввести свой номер телефона, к которому привязан Ваш аккаунт Telegram.
    * Получить и ввести код, пришедший по указанному номеру телефона (в сообщении мессенджера).

    Порядок работы:
1. Программа ожидает входящего сообщения от @EtherDROPS2_bot.
2. При поступлении нового сообщения проверяет его на соответсвие формату уведомления об изменении цены на криптовалюту.
3. Выделяет из письма необходимые данные: название монеты (криптовалюты); цену; % изменения цены и ссылку на веб-страницу биржи.
4. Так как страница биржи считается посредником, то начинается парсинг страницы и поиск ссылки на интересующий сайт: coinmarketcap.com (cmc)
5. Переход по найденной ссылке. Парсинг страницы монеты на "cmc".
6. Считывание данных о торговых парах из таблицы на сайте.
7. Отбор лучших торговых пар и вывод информации о них: ссылка на ориг. обменник; цена и объем продаж в %.

    Формат вывода программы:
1. Текст исходного сообщения из Telegram от @EtherDROPS2_bot (если это уведомл-е об измен-ии цены).
2. Перечень интересующей информации из письма: название монеты; ссылка на "cmc"; ссылка на "sushiswap"; цена в $ и % ее изменения.
3. *Здесь произойдет кратковременное открытие браузера для считывания таблицы торговых пар - закроется автоматически*
4. Вывод топа лучших (подход. по условиям относительно % изменения цены) торговых пар. Для каждой приводится: ссылка на ориг. обменник; цена в $ и объем продаж (в % и в $).
